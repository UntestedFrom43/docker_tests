docker run - старт контейнера из образа/ если образа нет локально, docker берет его в репозитории
docker ps - стата имеющихся запущенных контейнеров
docker ps -a - стата по всем имеющимся контейнерам
docker stop - остановка контейнера по имени или ID
docker rm - удаление контейнера тем же методом
docker images - список образов, установленных локально
docker rmi - удаление соответствующего образа/ важно иметь ввиду, что контейнер запущенный от этого образа установлен и удален
docker pull - скачиваем и сохраняем образ из репозитория без запуска
docker exec - исполнение команд на работающем контейнере
docker -d - запуск контейнера в фоновом режиме (detach)
docker attach - с указанием ID или имени выводит в консоль
